# fCC Learn npm package.json

A boilerplate for freeCodeCamp's Backend Challenges focused on learning Node.js project management and npm packages using `package.json`.

## Project Structure

- `server.js` - Express.js server entry point (do not edit per project instructions)
- `views/index.html` - Main HTML page
- `public/style.css` - Stylesheet
- `package.json` - Project metadata and dependencies

## Stack

- **Runtime**: Node.js
- **Framework**: Express.js v4
- **Package Manager**: npm

## Running

The app runs on port 5000 via the "Start application" workflow:

```
PORT=5000 node server.js
```

## Deployment

Configured for autoscale deployment running `node server.js`.
